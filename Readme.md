# Internet: entendendo os fundamentos da web

---

### Faça esse curso de Computação e:

> - Compreenda a evolução histórica e os fundamentos da internet

> - Identifique os tipos de redes e elementos da infraestrutura digital

> - Reconheça o papel dos protocolos **_HTTP_**, **_DNS_** e **_TCP_** na comunicação entre dispositivos

> - Explore a importância dos **_cabos submarinos_** e da **_arquitetura cliente-servidor_**

> - Utilize ferramentas como **_DevTools_** e **_Terminal_** para monitorar a performance dos sites.
