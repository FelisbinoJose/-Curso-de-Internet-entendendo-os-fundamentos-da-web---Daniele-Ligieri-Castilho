## Redes

- LAN (Local Area Network, rede local)
  - exemplo casa, escritório.

- MAN (Mentropolian Area Network, rede metropolitana)
  - exemplo cidade

- WAN (Wide Area Network, rede ampla, como a internet)
  - exemplo continente.
