# Arpanet: o nascimento da primeira rede

  - Agência de defesa voltada para pesquisa avançada em defesa.

  - Lançamento do satélite Sputnik.

  - Somente uso militar.

  - Conexão com quatro universidades (UCLA, Stanford, UCSB e Utah).
