## Como computadores representam informações?

  - Através pulsos elétricos e números binários.
----
## Entendo bits, bytes e múltiplos

  - Bits: 0 ou 1.
  - Bytes: 8 bits.

---

## O que são linguagens de programação?

  - Traduz linguagem humana em linguagem de computador.
  - Interpretador (traduz a linguagem depois de executar) x compilador (traduz antes de executar).
